Data Flow

1. User enters keywords → 2. FastAPI dispatches PubMed search via Celery → 3. Results stored in PostgreSQL → 4. Analysis engine computes bibliometrics (NetworkX, pandas, scikit-learn) → 5. Frontend renders interactive dashboard with D3.js networks + Recharts charts → 6. User filters, drills down, exports
